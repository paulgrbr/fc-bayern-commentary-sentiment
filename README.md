# fc-bayern-commentary-sentiment
Analyse möglicher Parteilichkeit in Fußball-Bundesligakommentaren des ZDF-Sportstudio-Kanals. Die Arbeit umfasst Datenerhebung über YouTube-Transkripte, umfangreiches Preprocessing mit LLMs, Segmentierung und Klassifikation der Kommentare sowie eine modellbasierte Sentimentanalyse über rund 7 800 kommentierte Einheiten.
